# wetwire-github-go

Declarative GitHub Actions workflows using native Go constructs.

## Overview

wetwire-github-go is a synthesis library that generates GitHub Actions workflow YAML from typed Go declarations. It follows the same patterns as [wetwire-aws-go](https://github.com/lex00/wetwire-aws-go).
